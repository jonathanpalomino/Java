¿Como hacer conexión con el servidor?
Para iniciar el Chat, asegúrese que en algún lado de la red (LAN o Internet), este siendo ejecutado el Servidor. Paso seguido ejecute el programa AplicacionChat.class, y le aparecerá lo siguiente:

Allí deberá indicar la dirección IP o nombre del equipo en donde se encuentra ejecutándose el servidor, y presionar Enter o el botón Aceptar. Esto hará que aparezca la interfaz gráfica del Chat:

Conectarse
Lo primero que hay que conocer es como conectarse, para ello vamos al menú 'Archivo' y presionamos 'Conectar'. entonces digitamos nuestro nick de Usuario y le damos 'Aceptar'.

Con ello aparecemos en la lista de la parte derecha, que es donde se muestran los últimos usuarios conectados; y se nos indica en la barra de estado que estamos conectados.

Enviar Mensajes
Para enviar mensajes solo tienes que digitarlos en la caja de texto que esta en la parte inferior y presionar Enter o el botón 'Enviar'; esto hará que se mande el mensaje al servidor y este lo muestre a todos los usuarios conectados a ese servidor. También puedes presionar cualquier Emoticon de la parte inferior con lo que se anexará una palabra clave que será interpretada por el programa para mostrar esa imagen.

Desconectarse
Para desconectarse vamos al menu 'Archivo' y presionamos 'Desconectar'. Con ello se cerrarán los flujos entre nosotros y el servidor, y aparecerá en la barra de estado lo siguiente:

Anexo: Información del desarrollo del programa
Casidiablo Chat surgió con la necesidad de crear una aplicación en Java, en donde se pudiera simular una Sala de Conversación o Chat. Esto, utilizando las principales características del paquete java.net de Java2, como lo son la implementación de Sockets y ServerSockets, para una comunicación tipo Cliente/Servidor. También utilizando el Subprocesamiento Múltiple o Hilos (Threads), para poder crear un ambiente Multitarea.

Este software fue desarrollado bajo el J2SDK1.4.1 de Java, y editado con el JCreator 3.50 de Xinox Software. También fue probado bajo diferentes Sistemas Operativos de Microsoft© y varias distribuciones de Linux como Knoppix y Red Hat, para resaltar así una de las principales características de Java: “La portabilidad”.
